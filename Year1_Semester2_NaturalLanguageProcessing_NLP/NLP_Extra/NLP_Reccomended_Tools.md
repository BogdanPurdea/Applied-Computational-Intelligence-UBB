# 🔖 Resources and Tools for Natural Language Processing (NLP)

## 📖 1. Text Pre-processing Pipeline
*Input: Raw Text*
1. **Tokenization** $\rightarrow$ tokens
2. **Sentence Segmentation** $\rightarrow$ sentences
3. **POS Tagging** $\rightarrow$ parts-of-speech
4. **Lemmatization** $\rightarrow$ lemmas
5. **NER (Named Entity Recognition)** $\rightarrow$ entities
6. **Chunking** $\rightarrow$ constituents, phrases
7. **Dependency Parsing** $\rightarrow$ binary grammatical relations
